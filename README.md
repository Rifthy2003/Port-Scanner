# 🔍 Port Scanner Tool

A simple Python-based port scanner to detect open/closed ports on a target machine.

## 🚀 Features
- Scans common ports (21, 22, 80, 443, 3306)
- Displays open/closed port status
- Easy to use command-line interface

## 🛠️ Tools Used
- Python
- socket library
